How to create a Teams bot
Option A: Teams Developer Portal (easiest)
1. Open Teams Developer Portal → Bot Management
2. Click + New Bot, give it a name, and click Add
3. Go to Configure — copy the Bot ID (this is your App ID)
4. Under Client secrets, click Add a client secret and copy the value
Option B: Azure Portal
1. Go to Azure Portal → Create Azure Bot
2. Create the bot, then go to Configuration to find the App ID
3. Click Manage Password → New client secret to get the App Password
After creating the bot, generate the endpoint below and paste it as the Messaging endpoint in your bot settings.
